# ProTrade Signal Bot - Free Hosting Guide

## Free Hosting Options

### Option 1: GitHub Pages (Recommended - FREE)

**Step 1: Create GitHub Account**
- Go to: https://github.com
- Sign up for free

**Step 2: Create Repository**
- Click **"+"** → **"New repository"**
- Name: `trade-signal-bot`
- Select **"Public"**
- Click **"Create repository"**

**Step 3: Upload Files**
- Click **"uploading an existing file"**
- Drag and drop all files from your `trade-signal-bot` folder
- Click **"Commit changes"**

**Step 4: Enable Pages**
- Go to **Settings** → **Pages** (left sidebar)
- Under **Build and deployment**:
  - Source: Select **"Deploy from a branch"**
  - Branch: Select **"main"** (or master)
  - Folder: Select **"/ (root)"**
- Click **Save**
- Wait 1-2 minutes

**Step 5: Access Your Site**
- Your site will be at: `https://yourusername.github.io/trade-signal-bot/`
- Share this link to access from phone!

---

### Option 2: Netlify (FREE Alternative)

**Step 1: Go to Netlify**
- https://app.netlify.com

**Step 2: Sign Up**
- Sign up with GitHub

**Step 3: Deploy**
- Drag your `trade-signal-bot` folder onto Netlify
- Done! Get your free URL
